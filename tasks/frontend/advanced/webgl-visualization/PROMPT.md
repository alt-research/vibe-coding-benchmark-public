# WebGL Data Visualization

Build high-performance WebGL visualization.

## Requirements
1. Render 1M+ data points at 60fps
2. Zoom/pan with level-of-detail
3. Hover/click interactions
4. Real-time data streaming
5. Multiple visualization types (scatter, heatmap, network)
6. Color scale and legend
7. Axis and grid rendering
8. Selection and brushing
9. Export to image
10. Responsive canvas sizing
