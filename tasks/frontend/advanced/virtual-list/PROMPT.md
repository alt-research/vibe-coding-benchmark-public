# Virtual List with Variable Heights

Build high-performance virtualized list component.

## Requirements
1. Variable row height support
2. Sticky headers/sections
3. Infinite scroll with data fetching
4. Smooth scrolling at 60fps
5. Keyboard navigation (arrow keys, page up/down)
6. Focus management for accessibility
7. Dynamic item loading
8. Scroll position restoration
9. Horizontal virtualization support
10. Memory-efficient for 100k+ items
