# Task: Mobile App Screen

## Objective
Build a mobile app screen using React Native/Expo from design specs.

## Screen: Product Detail
- Product image carousel
- Product title and price
- Size/color selectors
- Add to cart button
- Product description accordion
- Related products horizontal scroll

## Technical Requirements

1. **Components**
   - Image carousel with dots
   - Selector chips (size, color)
   - Quantity stepper
   - Expandable accordion
   - Horizontal scroll list

2. **Interactions**
   - Swipe through images
   - Tap to select size/color
   - Add to cart animation
   - Pull to refresh

3. **Styling**
   - Platform-specific styling (iOS/Android)
   - Safe area handling
   - Responsive to screen sizes

4. **Navigation**
   - Back button
   - Share button
   - Cart icon with badge

## Files to Create
- `screens/ProductDetail.tsx` - Main screen
- `components/ImageCarousel.tsx` - Image slider
- `components/SizeSelector.tsx` - Size picker
- `components/QuantityStepper.tsx` - Quantity control
- `components/Accordion.tsx` - Expandable section

## Success Criteria
- [ ] Matches design specs
- [ ] Carousel swipes smoothly
- [ ] Selectors work correctly
- [ ] Platform-appropriate styling
- [ ] Handles safe areas
