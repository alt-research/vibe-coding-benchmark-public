# Dynamic API Mocking Service

Build WireMock-style API mocking service.

## Requirements
1. Dynamic response generation
2. Stateful scenarios (sequences)
3. Request matching (body, headers, query)
4. Fault injection (delays, errors)
5. Record and playback
6. Contract verification
7. Template responses (Handlebars)
8. Webhook simulation
9. Admin API for configuration
10. Proxy mode with recording
