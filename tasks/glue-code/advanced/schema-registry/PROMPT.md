# Schema Registry Service

Build a schema registry for data contracts.

## Requirements
1. Avro, Protobuf, JSON Schema support
2. Compatibility modes (backward, forward, full)
3. Schema versioning and history
4. Automatic code generation
5. Schema validation API
6. Breaking change detection
7. Schema migration tooling
8. Integration with Kafka/Pulsar
9. Access control per schema
10. Schema diff visualization
