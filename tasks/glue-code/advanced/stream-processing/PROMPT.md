# Real-time Stream Processing

Build a real-time stream processing system with advanced features.

## Requirements
1. Tumbling, sliding, and session windows
2. Watermark generation for late data
3. Exactly-once processing semantics
4. Backpressure handling
5. State management with checkpointing
6. Dead letter queue for poison messages
7. Stream-table joins
8. Custom aggregation functions
9. Dynamic scaling based on lag
10. Metrics and alerting integration
