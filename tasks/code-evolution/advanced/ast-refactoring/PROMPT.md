# AST-Based Code Refactoring

Build automated code refactoring tool.

## Requirements
1. Parse multiple languages (TS, JS, Python)
2. Semantic-preserving transformations
3. Rename with scope awareness
4. Extract function/method
5. Inline variable/function
6. Convert patterns (async/await, etc.)
7. Comment preservation
8. Formatting preservation
9. Batch processing
10. Dry-run with diff preview
